  • Course- Structured Query Language (SQL) Duration- (15-20 Hours)

    Basic SQL Syntax:

    • SQL keywords (SELECT, FROM, WHERE, etc.)
    • Writing and executing SQL queries
    • Understanding tables and databases

    Retrieving Data:

    • SELECT statement and its various clauses (DISTINCT, ORDER BY, LIMIT)
    • Filtering data using WHERE clause
    • Sorting data using ORDER BY clause
    • Limiting the number of results using LIMIT clause
    • Aggregating data using GROUP BY clause

    Data Filtering and Manipulation:

    • Using logical operators (AND, OR, NOT) to combine conditions
    • Working with comparison operators (=, <, >, IN, BETWEEN)
    • Using wildcards (LIKE) for pattern matching
    • Performing calculations and transformations using arithmetic operators and SQL functions

    Joins and Relationships:

    • Understanding different types of joins (INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL JOIN)
    • Joining tables based on relationships (primary key and foreign key)
    • Handling NULL values in joins

    Subqueries and Derived Tables:

    • Writing subqueries to create more complex queries
    • Using subqueries in SELECT, WHERE, and FROM clauses
    • Creating derived tables and using them in queries

    Data Aggregation and Grouping:

    • Aggregating data using functions (SUM, AVG, MIN, MAX, COUNT)
    • Grouping data using GROUP BY clause
    • Filtering grouped data using HAVING clause

    Data Modification:

    • Inserting data into tables using INSERT statement
    • Updating existing data using UPDATE statement
    • Deleting data using DELETE statement

    Data Analysis Functions:

    • Using built-in SQL functions for data analysis (DATE, STRING, MATH functions)\
    • Working with aggregate functions (COUNT, SUM, AVG, etc.)
    • Performing calculations on date and time data

    Advanced Topics:

    • Working with multiple tables and complex joins
    • Indexing and optimizing queries for performance
    • Understanding database normalization and data modeling concepts
    • Time Series Analysis using SQL
